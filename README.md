## gitに変更を追加する
git add .

## gitに変更をコミットする
git commit

### gitのコミットのエディタを vscodeにする
git config --global core.editor 'code --wait'

### gitの変更をサーバーにPushする
git push

### サーバーから新しい変更を取得する
git pull
